# 一些eve online工具


1. 导入静态数据，执行import_sde_mongo.js(原始数据见有用链接)
2. 缓存市场数据，cache_market.js
3. 计算每种商品最大利润路线，calculate_type_profit.js, get_type_route.js
4. 计算指定两站之间的多商品运输利润路线，calculate_station_profit.js, get_station_route.js
